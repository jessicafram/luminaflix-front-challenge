# LuminaFlix Front Challenge

LuminaFlix é uma interface premium de streaming criada como desafio de front-end, com foco em hierarquia visual, polish de interface, organização de código e experiência de navegação.

## Tecnologias
- HTML5
- CSS3
- JavaScript Vanilla

## Funcionalidades
- Splash screen com animação letra por letra
- Hero section cinematográfica
- Perfis de usuário
- Galerias horizontais
- Favoritos com localStorage
- Ranking Top 10
- Destaque Top 3
- Busca por título e gênero
- Layout responsivo

## Estrutura
- `index.html`
- `css/`
- `js/`
- `assets/`
- `docs/`

## Como executar
Basta abrir o `index.html` no navegador ou usar a extensão Live Server no VS Code.
